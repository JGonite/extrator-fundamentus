#Extrator Fundamentus:

Esta aplicação acessa a página Fundamentus que contém uma extensa tabela de indicadores fundamentalistas da B3, extrai os dados dessa tabela utilizando a biblioteca JSoup e cria um arquivo .csv que contém as ações ordenadas por critério chamado *magic formula*, que indica quais são as melhores oportunidades da bolsa de valores (excluindo ações de energia, bancos e seguradoras).

para ler mais sobre a *magic formula*: https://www.investopedia.com/terms/m/magic-formula-investing.asp
